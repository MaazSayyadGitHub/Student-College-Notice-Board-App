# Student College (Notice) Board App
(This App is made For College Students And Admins To Handle Notices & To Be Update And Be Connected To Each Other Outside College As Well) 


### About App:-
#### Beutifull Home Design with Bottom Navigations And Navigation Drawers.
#### In This Student App Student can Login/Sign Up, See Updated/Recents Notices, with Title, Date And Times.
#### Can See Faculties Professors with Contact Information.
#### Can See Full Gallery For Festive Images And Memories with Zoom Image Feature.
#### About Section Of College.
#### Push Notification (FCM)
#### Sign In, Sign Up, Forget Password, And LogOut.
#### Can See PDFs For Reading/Downloading with Search Bar.
#### Themes, Video Lectures, 
#### This App is made with ( JAVA, Firebase Realtime Database, Xml, Material Design )
